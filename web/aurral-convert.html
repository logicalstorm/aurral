Spotify Import

Convert Exportify CSV files into Aurral playlist JSON. See the playlist import guide for accepted formats and import behavior.

  1. Log in to Exportify with your Spotify account.
  2. Click Export next to the playlist you want to bring to Aurral.
  3. Drop the downloaded .csv file into the box below and click Convert.
  4. Open Aurral, go to Playlists, and import the generated JSON.
