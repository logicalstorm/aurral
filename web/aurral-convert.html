Spotify ➜ Aurral

Public Playlists
  1. Paste a public Spotify playlist link, URI, or raw playlist ID.
  2. Click Fetch Playlist to build the Aurral JSON payload.
  3. Download the result and upload it in Aurral's Playlist Import modal.
Private Playlists
  1. Sign in to Exportify.net with Spotify.
  2. Export the playlist as CSV and drop it into the CSV import panel below.
  3. Download the converted JSON and import it in Aurral as usual.
