Aurral is the **Lidarr companion for music discovery**. It adds recommendations, requests, flows, and playlist downloads on top of your existing Lidarr stack, and keeps generated music out of your main library.

Use this page as a starting point for setup, day-to-day use, integrations, and administration.

## Recommended Stack

| Service                   | Role                                                        | Required               |
| ------------------------- | ----------------------------------------------------------- | ---------------------- |
| Lidarr                    | Library management, artist and album requests, queue status | Yes                    |
| Last.fm                   | Personalized discovery, tags, similar artists               | Recommended            |
| ListenBrainz              | Optional per-user listening history                         | No                     |
| Koito                     | Self-hosted per-user listening history                      | No                     |
| slskd                     | Soulseek-backed downloads for flows and playlists           | For Soulseek downloads |
| yt-dlp                    | YouTube/web fallback downloads for flows and playlists      | Default when installed |
| Prowlarr + SABnzbd/NZBGet | Usenet-backed downloads for flows and playlists             | For Usenet downloads   |
| Navidrome                 | Streaming generated playlists (M3U)                         | No                     |
| Plex                      | Flow playlists in Plexamp                                   | No                     |
| Ticketmaster              | Local concert recommendations                               | No                     |
