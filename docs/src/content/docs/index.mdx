Aurral is self-hosted music discovery for Lidarr-based stacks. It connects recommendations, requests, playlist downloads, and playback while keeping generated music separate from your main library.

Use this page as a starting point for setup, day-to-day use, integrations, and administration.

## Recommended Stack

| Service | Role | Required |
| --- | --- | --- |
| Lidarr | Library management, artist and album requests, queue status | Yes |
| Last.fm | Personalized discovery, tags, similar artists | Recommended |
| ListenBrainz | Optional per-user listening history | No |
| Koito | Self-hosted per-user listening history | No |
| slskd | Soulseek-backed downloads for flows and playlists | For downloads |
| Navidrome | Streaming generated playlists (M3U) | No |
| Plex | Flow playlists in Plexamp | No |
| Ticketmaster | Local concert recommendations | No |
