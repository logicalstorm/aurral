Aurral uses hosted metadata and search services for artist pages, album details, and catalog search. Fallbacks exist when those services are unavailable.

## Defaults

Most installs use Aurral's hosted metadata and search backends with no extra configuration.

## Custom providers

Admins can configure metadata hydration and the Aurral Search catalog backend in **Settings → Metadata Providers** (`/settings/metadata`).

Use custom providers when you self-host search infrastructure or need alternate metadata sources.

## Sponsorship

Hosted metadata and search run on personal infrastructure and may experience occasional downtime. [Sponsoring development](https://github.com/sponsors/lklynet/) helps keep those services reliable.
