Ticketmaster is optional. When configured, Aurral surfaces nearby shows for recommended, trending, and library-adjacent artists.

![Aurral nearby shows](../../../assets/screenshots/shows.webp)

## Setup

Add your Ticketmaster consumer key in onboarding or **Settings → Discover**.

Users can set location automatically or enter a ZIP or postal code from the Discover page when shows are enabled.
