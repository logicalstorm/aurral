---
title: Library
description: Browse artists already in Lidarr.
---

Library is a fast visual browser for artists already in Lidarr.

You can search, sort, open artist pages, and see which artists are monitored. Artist pages show library status, release groups, tags, similar artists, previews, and album actions.

When adding artists or albums from search or Discover, Aurral uses your Lidarr defaults unless you customize the add action.
